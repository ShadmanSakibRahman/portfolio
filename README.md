# Portfolio — Md. Shadman Sakib Rahman

Interactive 3D portfolio showcasing my work as a Data Engineer and AI/ML practitioner.

**Live:** [shadmansakibrahman.github.io/portfolio](https://shadmansakibrahman.github.io/portfolio/)

- Smooth scroll-triggered animations
- Fully responsive (mobile, tablet, desktop, ultrawide)
- Interactive project cards with live links
- Custom cursor effects
